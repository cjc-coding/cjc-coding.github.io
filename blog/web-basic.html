디지털 시대의 필수 언어인 코딩을 배우며 학생들은 논리력, 문제 해결 능력, 창의력을 함께 키웁니다. 12주간의 여정은 '나도 할 수 있다'는 자신감과 '만드는 즐거움'을 아이들에게 선물할 것입니다. 본 커리큘럼은 재미와 성취감이라는 두 마리 토끼를 모두 잡도록 설계되었습니다.

이론과 실습, 창의 활동의 균형 잡힌 분배를 통해 학습 효과를 극대화합니다.

핵심 개념: 웹사이트의 세 친구

학생들은 웹사이트를 만드는 세 가지 핵심 기술의 역할을 직관적인 비유를 통해 배웁니다. 이는 복잡한 개념을 쉽고 재미있게 기억하는 데 도움을 줍니다.

🦴

HTML (뼈대)

웹페이지의 구조와 콘텐츠를 담당합니다. 글, 이미지, 비디오 등이 들어갈 자리를 정해줘요.

👕

CSS (옷과 화장)

페이지를 예쁘게 꾸며줍니다. 색상, 글꼴, 레이아웃 등 디자인을 담당해요.

🧠

JavaScript (두뇌)

페이지를 동적으로 만듭니다. 버튼 클릭, 애니메이션 등 상호작용을 구현해요.

12주 상세 커리큘큘럼

"나만의 인터랙티브 포트폴리오 만들기" 최종 목표를 향해, 매주 체계적인 학습과 실습을 진행합니다.

주차 핵심 목표 주요 학습 내용 실습 활동
1주차 코딩과 첫 만남 웹 동작 원리 이해, HTML/CSS/JS의 역할, 기본 태그 (`<h1>`, `<p>`) 나의 이름과 다짐을 담은 첫 웹페이지 만들기
2주차 콘텐츠 풍성하게 채우기 이미지(`<img>`), 링크(`<a>`), 목록(`<ul>`, `<li>`) 태그 내가 좋아하는 것들(이미지, 영상 링크)로 페이지 채우기
3주차 CSS로 페이지 꾸미기 CSS 기본 문법, `<style>` 태그, `color`, `background-color` 페이지 배경색과 글자색을 내가 좋아하는 색으로 변경하기
4주차 폰트와 여백 다루기 `font-size`, `text-align`, 박스모델 기초 (`margin`, `padding`) 글자 크기, 정렬을 조절하고 요소 간 여백 주기
5주차 체계적인 스타일 관리 `class`와 `id` 선택자, `<div>`로 영역 나누기 `class`를 이용해 특정 부분만 다르게 꾸미기
6주차 Flexbox로 레이아웃 잡기 `display: flex`, `justify-content`, `align-items` 속성 프로필 사진과 소개글을 가로로 나란히 배치하기
7주차 JavaScript와 첫 인사 JS의 역할, `onclick` 이벤트, `alert()` 함수 클릭하면 "반가워!" 메시지 창이 뜨는 버튼 만들기
8주차 웹페이지 내용 바꾸기 변수(let), `getElementById()`, `innerHTML` 속성 버튼 클릭 시 페이지의 글자 내용이 바뀌는 기능 구현
9주차 조건문으로 똑똑하게 만들기 `if / else` 조건문, 간단한 비교 연산자 정답/오답을 알려주는 나만의 퀴즈 만들기
10주차 함수와 스타일 제어 함수(function) 개념, JS로 CSS 스타일 변경하기 '다크/라이트 모드' 전환 버튼 기능 구현하기
11주차 최종 프로젝트 기획 및 제작 아이디어 구상, 기능 설계, 배운 내용 총 복습 자신만의 포트폴리오/미니게임 기획 및 기본 구조 제작
12주차 프로젝트 완성 및 발표 버그 수정, 디자인 다듬기, 발표 준비 프로젝트 완성 및 친구들 앞에서 작품 시연, 공유하기

예상 학습 성장 곡선 (12주)
