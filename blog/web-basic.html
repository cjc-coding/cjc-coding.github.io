코딩, 미래를 여는 열쇠

디지털 시대의 필수 언어인 코딩을 배우며 학생들은 논리력, 문제 해결 능력, 창의력을 함께 키웁니다. 12주간의 여정은 '나도 할 수 있다'는 자신감과 '만드는 즐거움'을 아이들에게 선물할 것입니다. 본 커리큘럼은 재미와 성취감이라는 두 마리 토끼를 모두 잡도록 설계되었습니다.

🚀 💡 🎨

이론과 실습, 창의 활동의 균형 잡힌 분배를 통해 학습 효과를 극대화합니다.

핵심 개념: 웹사이트의 세 친구

학생들은 웹사이트를 만드는 세 가지 핵심 기술의 역할을 직관적인 비유를 통해 배웁니다. 이는 복잡한 개념을 쉽고 재미있게 기억하는 데 도움을 줍니다.

🦴

HTML (뼈대)

웹페이지의 구조와 콘텐츠를 담당합니다. 글, 이미지, 비디오 등이 들어갈 자리를 정해줘요.

👕

CSS (옷과 화장)

페이지를 예쁘게 꾸며줍니다. 색상, 글꼴, 레이아웃 등 디자인을 담당해요.

🧠

JavaScript (두뇌)

페이지를 동적으로 만듭니다. 버튼 클릭, 애니메이션 등 상호작용을 구현해요.

12주간의 코딩 모험 로드맵

"나만의 인터랙티브 포트폴리오 만들기"를 목표로, 매 2주마다 새로운 기술을 습득하고 적용하며 점진적으로 실력을 쌓아갑니다.

1-2주

HTML 기초

페이지 뼈대 세우기, 콘텐츠 채우기

3-4주

CSS 첫걸음

색깔, 폰트, 꾸미기

5-6주

CSS 레이아웃

요소 정렬, 반응형 기초

7-8주

JS 맛보기

클릭 이벤트, 간단한 반응

9-10주

JS 활용

DOM 조작, 동적 기능

11-12주

프로젝트 완성

자유 창작 및 발표

예상 학습 성장 곡선 (12주)

초반의 흥미가 중반의 도전 과제를 거치며 완만하게 성장하다, 프로젝트가 완성되는 후반부에 자신감과 실력이 급격히 상승합니다.
